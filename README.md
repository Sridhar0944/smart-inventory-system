# 📦 Smart Inventory Management System

A full-stack Inventory Management System built using React, TypeScript, Node.js, Express.js, and MySQL.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 📊 Dashboard Analytics
- 📦 Product Management
- 🏷️ Category Management
- 🚚 Supplier Management
- 📥 Stock In
- 📤 Stock Out
- ⚠️ Low Stock Alert
- 🖼️ Product Image Upload
- 📱 QR Code Generation
- 📊 Barcode Generation
- 📄 Reports
- 🔍 Search & Filter
- 📱 Responsive Design

---

## 🛠️ Tech Stack

### Frontend

- React.js
- TypeScript
- Tailwind CSS
- React Router
- Axios
- React Hook Form
- Zod
- React Hot Toast

### Backend

- Node.js
- Express.js
- MySQL
- JWT Authentication
- Multer

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sridhar0944/smart-inventory-system.git
```

### Frontend

```bash
cd client
npm install
npm run dev
```

### Backend

```bash
cd server
npm install
npm start
```
